Defuse It!

How to play?

"Defuse It!" includes 4 Mini-Games.
Complete them all to defuse the bomb.
The sooner you finish, the better score you get.
*DO NOT refresh the browser while playing the game*
Good luck and have fun!

Mini-Game 1
Number Of The Day

Guess the correct number from the interval given


Mini-Game 2
Wordle?

Guess the correct word from the letters given

Mini-Game 3
Code breaker!

Solve the 3 stage mystery code



Mini-Game 4
Defuse It!

Write the final code which includes previous answers in correct order
